# AgroRega Manager Demo

Webapp demo estática para gestão de empresa de instalação de rega agrícola.

## Deploy no GitHub Pages

1. Criar um repositório novo no GitHub, por exemplo `agrorega-demo`.
2. Enviar todos os ficheiros desta pasta para a raiz do repositório.
3. No repositório, ir a **Settings > Pages**.
4. Em **Build and deployment**, escolher:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
5. Guardar.

O site ficará disponível em:

`https://TEU-UTILIZADOR.github.io/agrorega-demo/`
