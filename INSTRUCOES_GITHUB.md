# Instruções rápidas para publicar

## Método sem terminal

1. Vai a github.com e cria um repositório chamado `agrorega-demo`.
2. Entra no repositório.
3. Clica em `Add file > Upload files`.
4. Arrasta para lá todos os ficheiros desta pasta, não o ZIP fechado.
5. Clica em `Commit changes`.
6. Vai a `Settings > Pages`.
7. Em `Source`, escolhe `Deploy from a branch`.
8. Em `Branch`, escolhe `main` e `/root`.
9. Clica em `Save`.

O GitHub pode demorar alguns minutos a publicar.
